# Getting Started

Component Driven Redux Framework is a framework for building React applications with Redux. It is designed to help you build your application with a component driven approach. It is built on top of Vite, React, Redux, and React Router. TypeScript is used throughout the framework. It is designed to be flexible and extensible. It is also designed to be easy to use and easy to learn. The framework is designed to be used with the following tools:

- [Vite](https://vitejs.dev/)
- [React](https://reactjs.org/)
- [Redux](https://redux.js.org/)
- [React Router](https://reactrouter.com/)
- [TypeScript](https://www.typescriptlang.org/)
- [Jest](https://jestjs.io/)
- [React Testing Library](https://testing-library.com/docs/react-testing-library/intro/)
- [ESLint](https://eslint.org/)
- [Redux Toolkit](https://redux-toolkit.js.org/)
- [React Redux](https://react-redux.js.org/)
